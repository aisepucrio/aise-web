AnnotAISE — Human-in-the-loop Annotation Platform (AISE Lab)

One-liner. A form-driven platform that replaces messy spreadsheets with guided workflows for creating, distributing, and auditing large-scale manual annotations.

Problem. Teams validating datasets at scale often fall back to spreadsheets shared over chat/email. That creates version drift, poor UX for labelers, no quality control, and painful progress tracking.

What AnnotAISE does.

Admin space. Create projects and “labeling campaigns”, design multi-section forms (contexts + questions), define label taxonomies, assign roles (admin/labeler), set deadlines, and watch live progress (done/pending/late).

Labeler space. Clean UI with keyboard-friendly forms, contextual panels, per-page progress, confidence capture, and multi-label support (e.g., emotions).

QA & analytics. Inter-annotator agreement (Cohen’s κ), disagreement heatmaps, reviewer queues, SLA/latency metrics, backlog burn-down, export pipelines.

Governance. Role-based access, audit logs, versioned guidelines, and dataset exports with provenance.

Tech (reference implementation). Next.js + TypeScript + Mantine UI, NestJS API, PostgreSQL + Prisma, JWT auth, object storage for datasets, telemetry via OpenTelemetry.
